# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行美国手语手势检测
- 支持选择不同检测模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 美国手语识别与学习
- 辅助听障人士交流
- 餐饮消费场景的手语识别
- 英文字母手语识别
- 人机交互研究与应用

## 技术特点
- 基于YOLOv8深度学习框架构建
- 使用PyQt5开发图形用户界面
- 内置9990张标注数据集，涵盖26个英文字母及餐饮消费场景的120类词汇
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 预训练模型可直接使用，无需重新训练
- 环境配置包括Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9等
